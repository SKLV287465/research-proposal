\section{Operating Systems}
The operating system is the lowest level of software directly interfacing with hardware and providing user applications a general, usable interface to operate upon. Essentially, it is an abstract machine and a resource manager working with \texttt{policies} to multiplex and allocate hardware resources through a simplified system call interface. \citep{keylist}
